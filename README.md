# Expense Tracker - Console Application

A console-based expense tracking application built in C# (.NET 8).

## Features
- Add expenses with title, amount, category, and date
- View all recorded expenses
- Filter expenses by category
- Calculate total spending
- Data persistence using file storage (no database required)

## Concepts Used
- Object-Oriented Programming (Classes, Properties, Methods)
- Collections (List<T>)
- LINQ (Where, Sum)
- File I/O (Read/Write)
- Exception-safe input handling
- Control flow (while loop, switch statement)

## How to Run
1. Clone this repository
2. Open in Visual Studio 2022
3. Press F5 to run
